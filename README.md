# Ingeniería Informática

Portfolio académico del Grado en Ingeniería Informática.

Este repositorio recoge proyectos y prácticas desarrolladas a lo largo de la carrera, enfocados en programación, sistemas distribuidos, bases de datos y desarrollo web.

## Contenidos
- Proyectos de programación en Java y Python
- Sistemas distribuidos y comunicación entre servicios
- Bases de datos y persistencia
- Desarrollo web y APIs

## Objetivo
Mostrar de forma organizada los conocimientos adquiridos durante la carrera y servir como portfolio académico y profesional.

## Tecnologías
- Java
- Python
- JavaScript
- SQL
- Git y GitHub
